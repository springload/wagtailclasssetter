wagtailclasssetter
==================

Class setter for Wagtail Hallo.js editor

# Quickstart

``` $ pip install wagtailclasssetter [GITHUB SSH URI]```

add wagtailclasssetter to your settings.py in the INSTALLED_APPS section:

```
...
    'modelcluster',
    'wagtailclasssetter',
    'core',
    'wagtail.contrib.wagtailsitemaps',
...
```
